Farklı Python Sürümleri ile Çalışmak

Posted on Pzt 26 Ağustos 2019 in Python3

Merhaba arkadaşlar, bu yazımda bilgisayarımızdaki farklı Python sürümleri ile çalışmanın nasıl olduğunu göstereceğim.

Peki ya neden farklı Python sürümleri ile çalışalım?

Bunun birden fazla yanıtı var. Benim kullanım sebebim bazı modüller Python 3.5 ve altını destekliyor ancak ben kendi projelerimde 3.7'de kullanmak istiyorum. Yani hem 3.5 ile hem 3.7 ile projemi test edip çalıştırıyorum.

Bilgisayarımızdaki Python Sürümlerini Görüntüleme

Şimdi gelin bilgisayarımızda hangi Python sürümleri kurulu görelim :)

ls /usr/bin/python*

Yukarıdaki komutu yazdıktan sonra bilgisayarınızda kurulu olan Python sürümlerini görüntüleyebilirsiniz.

Farklı Python Sürümü Kurmak

Gelin şimdi istediğimiz Python sürümünü indirelim sonrsında onu kuralım. İlk olarak bu adresten istediğimiz Python sürümünü .tar.xz uzantısı ile indirilerim. Bu indirme işini terminali açıp

wget https://www.python.org/ftp/python/3.5.3/Python-3.5.3.tar.xz

indirebiliriz. Ardından komut satırından indirdiğimiz dizine girip

tar xf Python-3.5.3.tar.xz

yazarak dosyamızı çıkartma işlemini yapabiliriz. Şimdi çıkan dosyamızın içine girelim

cd Python-3.5.3

Ardından sırasıyla aşağıdaki iki komutu çalıştıralım,

sudo ./configure --enable-optimizations

sudo make altinstall

Komutların tamamlanması uzun sürebilir sabırlı olmakta fayda var :) Bittikten sonra artık yeni Python sürümünüz kurulmuş olacaktır.

Farklı Sürümler ile Çalışalım

Gelelim şimdi asıl konumuza. Bir projemde 3.5 başka bir projemde 3.7 kullanmak istiyorum ne yapmalıyım? Aslında oldukça kolay. Gelin iki tane proje açalım. Sırasıyla aşağıdaki komutları terminalde çalıştırın.

cd
mkdir Python35Proje
cd Python35Proje
python3.5 -m venv myenv
source myenv/bin/activate

Şimdi aynı yerde bir dosya açalım ve içine aşağıdaki kodları yazıp çalıştıralım.

import platform
print(platform.python_version())

Kodu çalıştırdığınızda eğer benimle aynı adımları izlediyseniz 3.5.3 yani sanal ortamı kurduğumuz Python sürümü ekrana yazılacaktır. Ardından deactivate yazarak sanal ortamdan çıkış yapalım.

Şimdi başka bir proje açalım ve bunda başka bir sürüm kullanalım.

cd
mkdir Python37Proje
cd Python37Proje
python3.7 -m venv myvenv
source muvenv/bin/activate

Şimdi aynı yerde bir dosya açalım ve içine aşağıdaki kodları yazıp çalıştıralım.

import platform
print(platform.python_version())

Kodu çalıştırdığınızda 3.7.x yazacaktır. Yani sanal ortamı kurduğunuz Python3.7 sürümü hangisi ise o yazacaktır. Aynı şekilde deactivate yazarak sanal ortamı kapatabilirsiniz.

Bu yazımızı burada sonlandıralım arkadaşlar. Umarım faydalı olmuştur.

Herkese iyi çalışmalar diliyorum :)
